蓮花塔一對式客製化選購 v3

使用方式：
1. 解壓縮 ZIP。
2. 直接用瀏覽器開啟 index.html。
3. assets 資料夾內是已做基本去背的 PNG 圖片。

本版修正：
- 放入 4 種底座、6 種上方造型。
- 選擇底座與造型可即時切換。
- 清空表單、重設全部可正常使用。
- 圖片已做基本去背處理，適合測試圖層疊合。

注意：
部分原圖是白底或假透明棋盤格背景，已用程式做基本去背。
如果要正式上線，建議之後再逐張精修去背邊緣，效果會更漂亮。


v6 調整說明：
1. 上方造型已稍微往下修。
2. 每個底座可單獨調整高度，不改寬度。
   請打開 index.html，搜尋 baseList。
   每個底座都有 baseH，例如：
   "baseH": 1
   改成：
   "baseH": 1.15
   就是高度拉高 15%，寬度不變。
   改成：
   "baseH": 0.9
   就是高度壓低 10%，寬度不變。

3. 上方造型位置請搜尋 .top-img：
   bottom:var(--join, 55%);
   數字越大，造型越往上。
   數字越小，造型越往下。

   transform:translateY(var(--overlap, 3%)) scale(var(--scale,0.8));
   translateY 數字越大，造型越往下。
   scale 0.8 代表整體縮小為 80%。
